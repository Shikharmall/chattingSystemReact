# Responsive Chatting System (React JS(VITE) + Tailwind CSS)

### Clone Project

   npm clone https://github.com/Shikharmall/chattingSystemReact.git
   
   npm install
   
   npm run dev

### Features

  Users can send messages in form of text and images.
  
  Users can serach for other users.
  
  Clear whole chat.
